# fastapi-cas-example

An example repo to demo how to integrate python-cas into a fastapi app.

# run

```shell
git clone https://github.com/Chise1/fastapi-cas-example
cd fastapi-cas-example
pip3 install poetry
poetry install
#in python virtualenvs
uvicorn run main.py
```
